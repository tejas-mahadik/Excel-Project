## 📋 Project Overview

This project is an **interactive Excel dashboard** built on Blinkit's grocery sales dataset, providing a comprehensive visual analysis of outlet performance, item types, fat content distribution, ratings, and location-based sales trends — inspired by Blinkit, *India's Last Minute App*.

---

## 📁 Project Files

| File | Description |
|------|-------------|
| `data_of_blinkit.xlsx` | Raw dataset containing Blinkit grocery sales records |
| `Blinkit_Project.xlsx` | Excel workbook with cleaned data and interactive dashboard |
| `Blinkit_Dasboard_image.PNG` | Dashboard screenshot/preview |

## 📊 Dashboard Highlights

### KPI Cards
| Metric | Value |
|--------|-------|
| Total Sales | $1.20M |
| Avg Sales | $141 |
| No. of Items | 8,523 |
| Avg Rating | 4.0 |

### Charts & Visuals

- **Donut Chart (Fat Content)** — 65% Regular fat ($776.3K) vs. 35% Low Fat ($425.4K) items
- **Bar Chart (Fat by Outlet)** — Sales split by fat content across Tier 1, 2, and 3 outlets
- **Horizontal Bar Chart (Item Type)** — Top categories: Fruits & Vegetables ($178.1K), Snack Foods ($175.4K), Household ($136K)
- **Line/Area Chart (Outlet Establishment)** — Sales trend from 2011 to 2022; peak at 2018 with $204.5K
- **Donut Chart (Outlet Size)** — High: 42% ($507.9K), Medium: 21% ($249.0K), Small: 37% ($444.8K)
- **Bar Chart (Outlet Location)** — Tier 3 leads with $472.1K, followed by Tier 2 ($393.2K) and Tier 1 ($336.4K)
- **Table (Outlet Type)** — Supermarket Type1 dominates in Total Sales ($787.5K), with Grocery Store having the most items (1,083)

### Filter Panel (Slicers)
- Filter by **Outlet Size** (High / Medium / Small)
- Filter by **Outlet Location** (Tier 1 / Tier 2 / Tier 3)
- Filter by **Item Type** (Health & Hygiene, Household, Meat, Others, Seafood, etc.)

## 🗂️ Dataset Details

**Source:** Blinkit Grocery Sales Dataset

| Column | Description |
|--------|-------------|
| `Item Fat Content` | Low Fat or Regular |
| `Item Identifier` | Unique item code |
| `Item Type` | Category (Fruits & Vegetables, Snack Foods, Dairy, etc.) |
| `Outlet Establishment Year` | Year the outlet was established (2011–2022) |
| `Outlet Identifier` | Unique outlet code |
| `Outlet Location Type` | Tier 1, Tier 2, or Tier 3 city |
| `Outlet Size` | High, Medium, or Small |
| `Outlet Type` | Supermarket Type1/2/3 or Grocery Store |
| `Item Visibility` | Product shelf visibility score |
| `Item Weight` | Weight of the item |
| `Sales` | Sales amount in USD |
| `Rating` | Customer rating (out of 5) |

## 🔍 Key Insights

1. **Total sales crossed $1.20M** across all outlets with an average transaction value of $141.
2. **Regular fat items** outsell Low Fat items — contributing 65% of total sales ($776.3K).
3. **Tier 3 locations** generate the highest revenue ($472.1K), suggesting strong demand in smaller cities.
4. **Fruits & Vegetables and Snack Foods** are the top-performing item categories.
5. **2018** was the peak year for outlet sales with $204.5K, showing a significant growth phase.
6. **Supermarket Type1** outlets account for the majority of total sales ($787.5K).
7. **Average customer rating** stands at a healthy **4.0 out of 5**.

## 🚀 How to Use

1. Open `Blinkit_Project.xlsx` in Microsoft Excel (2016 or later recommended).
2. Navigate to the **Dashboard** sheet.
3. Use the **Filter Panel** on the left to slice data by Outlet Size, Location, or Item Type.
4. All KPIs and charts update dynamically based on your filter selections.
