📊 Data Analysis in Excel
Welcome to the Data Analysis in Excel repository! This collection features comprehensive Excel-based data analysis projects, each tailored to a specific industry or dataset. These projects highlight the power of Microsoft Excel in transforming raw data into actionable insights through dashboards, charts, and pivot-based summaries.

1️⃣ Blinkit Grocery Sales Dashboard
Show Image
An interactive Excel dashboard built on Blinkit's grocery sales dataset — India's Last Minute App. Key features include:

KPI Summary: Total Sales ($1.20M), Average Sales ($141), Total Items (8,523), and Average Rating (4.0)
Fat Content Analysis: Donut chart comparing Low Fat vs. Regular item sales
Item Type Breakdown: Sales performance across 16+ product categories including Fruits & Vegetables, Snack Foods, and Household items
Outlet Establishment Trend: Year-wise sales trend from 2011 to 2022
Outlet Size & Location: Performance comparison across High/Medium/Small outlets and Tier 1/2/3 cities
Outlet Type Comparison: Total Sales, Avg Sales, and No. of Items across Supermarket types and Grocery Stores

This dashboard is a powerful tool for analyzing grocery retail performance and identifying high-revenue outlet segments.

2️⃣ Netflix Content Dashboard
Show Image
A comprehensive Excel dashboard analyzing Netflix's content library of 8,392 titles. Key features include:

KPI Summary: Total Titles (8,392), Total Movies (5,748), Total TV Shows (2,643), Latest Release Year (2021)
Content Type Distribution: Donut chart comparing Movies vs. TV Shows
Yearly Trend: Area chart showing content additions from 2012 to 2021, peaking in 2018
Country-Wise Analysis: Bar chart of top content-producing countries — United States leads with 2,630 titles
Rating Breakdown: Horizontal bar chart across all ratings — TV-MA dominates with 3,154 titles
Filter Panel: Interactive slicers for Type, Rating, and Country

This dashboard provides valuable insights into Netflix's content strategy and global production trends.
